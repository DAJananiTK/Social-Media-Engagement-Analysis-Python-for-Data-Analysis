# 📊 Social Media Engagement Analysis using Python

## 📌 Project Overview

This project analyzes a social media engagement dataset using Python to understand user behavior, content performance, audience demographics, and engagement trends. The dataset was cleaned, explored, statistically analyzed, and visualized to extract meaningful insights that can support data-driven decision-making for social media strategies.

---

## 🎯 Objectives

- Import and preprocess the dataset using Python.
- Clean missing values, duplicates, and incorrect data types.
- Perform Exploratory Data Analysis (EDA).
- Generate descriptive statistics and correlation analysis.
- Create meaningful visualizations using Matplotlib, Seaborn, and Plotly.
- Identify trends in user engagement, sentiment, content performance, and audience behavior.
- Provide business insights based on the analysis.

---

## 📈 Project Workflow

### Task 1 – Data Import & Setup
- Imported the CSV dataset using Pandas.
- Checked and corrected data types.
- Converted date columns to datetime format.

### Task 2 – Data Cleaning
- Detected and handled missing values.
- Removed duplicate records.
- Standardized categorical values.
- Corrected data formatting.
- Created a new feature for hashtag count.

### Task 3 – Exploratory Data Analysis
- Explored dataset structure.
- Generated summary statistics.
- Analyzed categorical distributions.
- Created correlation matrix.
- Performed GroupBy analysis.

### Task 4 – Data Wrangling
- Created engagement-related features.
- Performed grouped summaries by:
  - Post Type
  - Country
  - Sentiment

## 📊 Key Findings

- Video posts achieved the highest engagement rate.
- Food was the best-performing content category.
- Brazil recorded the highest average engagement rate.
- Verified accounts generated slightly higher engagement than non-verified accounts.
- Users ranged from 13 to 64 years, with an average age of 38 years.
- Mobile devices recorded the highest average watch time.
- Negative sentiment posts achieved the highest engagement rate.
- Most numerical variables showed weak correlations, indicating that engagement depends on multiple factors.
